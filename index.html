Process Analysis

Project
Goal

To test my skills and help develop my expereince with React, i chose to Design and build this GPT-3 website clone in React based on the figma documentation. I really learned alot through this project and I would love to take you on my journey.

Final
Product

Step One

The first step to build any site replica is downloading the assets and setting up the basic file structure of the project. Thankfully the figma documentation also came with a google drive link for all the assets used in the design prototype.

Step Two

Now that the file structure is setup the next step was creating the basic element of the website such as the nav bar and header. To do so it was a combination of experimentation and google-fu to get the results needed to replicate the design

Step Three

Now that the nav bar is in place, the next piece was building the header content as all the content following would carry a very similar structure. This part was the hardest to get the spacing right especiialy when in responsive mode. Alot of trial and error got it to where I was happy with the outcome and i believed it to be close enough. The difficulty here was not with the React code but rather the CSS structure to make it look right.

Final Step

Rather than repeat what I showed above much rest of the project was very similar in terms of scope and how to create the rest of the elements. The next obstance was importing the page into the React file and making it all work together. This is the most consuming part for me. I had to import all the indivudal jsx files into the DOM and render them in the React viewport.
